## üöÄ Why this system?

Most systems fail because they require too much willpower (tagging, folder sorting, metadata) *before* you write. Synapse is designed to be **frictionless**.

*   **Capture Instantly:** A dedicated Inbox and Daily Log means you never wonder "Where do I put this?"
*   **Atomic Thinking:** Break big projects into small, linkable "Atomic Notes."
*   **Native Performance:** Relies on core Obsidian features (Search, Links, Graph). **No heavy plugins required.**
*   **Scalable:** Works for 10 notes or 10,000 notes without changing the structure.

## ‚ö° How to Use (The Workflow)

### 1. Capture (Daily)
Don't organize. Just write.
*   **Quick thoughts?** Put them in your **Daily Note** (`10 Daily`).
*   **Files/Drafts?** Drop them in **Inbox** (`00 Inbox`).

### 2. Process (Weekly)
Once a week, empty your Inbox.
*   **Tasks** -> Move to your To-Do app (e.g., TickTick, Todoist).
*   **Knowledge** -> Refine into an **Atomic Note** in `30 Atomic Notes`.
*   **Trash** -> Delete the noise.

### 3. Connect (Always)
Never leave a note alone.
*   Don't use folders to group things.
*   Use **Links** (`[[ ]]`).
*   *Example:* "This note about [[Habits]] reminds me of [[Dopamine]]."

## üì• Installation

1.  **Download** this repository as a ZIP file (or clone it).
2.  **Open Obsidian**.
3.  Select **"Open folder as vault"**.
4.  Choose the unzipped folder.
5.  **Start Writing.** (Read `START HERE - System Guide.md` inside the vault for a walkthrough).

*(Note: This vault comes with a few **Example Files** about the "Zeigarnik Effect" to show you how the connections work. Feel free to explore them and then **delete them** when you are ready to start fresh.)*

### Recommended Settings
*   **Files & Links > Default location for new notes:** `00 Inbox`
*   **Files & Links > Default location for attachments:** `98 Attachments`
*   **Core Plugins > Templates > Template folder location:** `99 Templates`
